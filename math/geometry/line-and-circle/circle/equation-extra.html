圆的参数方程​

圆心为 (x0,y0)(x_0, y_0)(x0​,y0​)，半径为 rrr 的圆，可表示为含参数 θ\thetaθ 的方程组：

{x=x0+rcos⁡θy=y0+rsin⁡θ\bcs x = x_0 + r\cos \theta \\ y = y_0 + r\sin \theta \ecs{x=x0​+rcosθy=y0​+rsinθ​

值得注意的是，其形式与直线的参数方程看起来一致（只是字母从 ttt 变成了 rrr），但有本质的不同：

  • 直线方程中 θ\thetaθ 为 常量（代表倾斜角），而 ttt 为 参数．
  • 这里 rrr 为 常量（代表直径），而 θ\thetaθ 为 参数．

圆的参数方程相比于标准方程，从二次变成了一次，引入了三角函数．善用圆的参数方程可能可以达到简便计算的目的．

圆的直径式​

已知坐标的两点 AAA，BBB，求以 ABABAB 为直径的圆的方程．

可以直接刻画为圆心在 ABABAB 中点，且半径为 ∣AB∣2\df{|AB|}22∣AB∣​，列标准方程，但有一个更简单的做法——圆的直径式：

直径端点分别为 A(x1,y1)A(x_1, y_1)A(x1​,y1​) 和 B(x2,y2)B(x_2, y_2)B(x2​,y2​) 的圆可以表示为

(x−x1)(x−x2)+(y−y1)(y−y2)=0(x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0(x−x1​)(x−x2​)+(y−y1​)(y−y2​)=0
证明

考虑平面直角坐标系中任意一点 P(x,y)P(x, y)P(x,y)．

  • 当它与 AAA 或与 BBB 重合时，显然 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP（因为其中一个是零向量）．
  • 否则，它在圆上等价于 ∠APB=π2\angle APB = \df \pi 2∠APB=2π​，即 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP．
  • 可以证明，PPP 不在圆上时，∠APB≠π2\angle APB \ne \df \pi 2∠APB=2π​．

因此 PPP 在圆上等价于 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP，即

(x−x1,y−y1)⋅(x−x2,y−y2)=0(x - x_1, y - y_1) \cdot (x - x_2, y - y_2) = 0(x−x1​,y−y1​)⋅(x−x2​,y−y2​)=0

将数量积展开后，即得目标方程．

公共弦所在直线方程​

在两圆 ⊙A\odot A⊙A，⊙B\odot B⊙B 恰交于 PPP，QQQ 两点时，称线段 PQPQPQ 为两圆的 公共弦．现在研究直线 PQPQPQ 的方程．

有结论：将两个圆方程的 一般式 形式 相减，会得到一个 一次方程，这个方程就是两圆的公共弦方程．

证明

设 ⊙A\odot A⊙A 的一般式为

x2+y2+D1x+E1y+F1=0x^2 + y^2 + D_1x + E_1y + F_1 = 0x2+y2+D1​x+E1​y+F1​=0

⊙B\odot B⊙B 的一般式为

x2+y2+D2x+E2y+F2=0x^2 + y^2 + D_2x + E_2y + F_2 = 0x2+y2+D2​x+E2​y+F2​=0

两方程相减得

(D1−D2)x+(E1−E2)y+F1−F2=0(D_1 - D_2)x + (E_1 - E_2)y + F_1 - F_2 = 0(D1​−D2​)x+(E1​−E2​)y+F1​−F2​=0

两圆相交于两点，则圆心不可能重合，因此 D1=D2D_1 = D_2D1​=D2​ 与 E1=E2E_1 = E_2E1​=E2​ 不可能同时成立，因此 D1−D2≠0D_1 - D_2 \ne 0D1​−D2​=0 或 E1−E2≠0E_1 - E_2 \ne 0E1​−E2​=0．得到的方程表示一条直线．

考虑两圆的交点 PPP 和 QQQ．由于它们各自都 同时满足两个圆的方程，因此也一定满足 两个圆方程相减得到的方程．即 PPP 和 QQQ 在表示的直线上，两点确定一条直线，证毕．

例题 1.1

求两圆 x2+y2−2x+10y−24=0x^2 + y^2 - 2x + 10y - 24 = 0x2+y2−2x+10y−24=0 与 x2+y2+2x+2y−8=0x^2 + y^2 + 2x + 2y - 8 = 0x2+y2+2x+2y−8=0 的公共弦所在直线方程．

例题 1.1 解答

两方程相减得 −4x+8y−16=0-4x + 8y - 16 = 0−4x+8y−16=0，整理得 x−2y+4=0x - 2y + 4 = 0x−2y+4=0．

两圆必须相交

当两圆的交点数量不为 222 时，不存在 公共弦，两圆相减得到的方程不再是公共弦．

不过比较值得注意的是，当两圆交点数量为 111，即 相切（包含内切与外切）时，会得到两个圆在 切点 上的一条 公切线．

证明

两圆圆心 A(−D12,−E12)A(-\df{D_1} 2, -\df{E_1} 2)A(−2D1​​,−2E1​​)，B(−D22,−E22)B(-\df{D_2}2, -\df{E_2}2)B(−2D2​​,−2E2​​)，有

AB→=(D1−D22,E1−E22)=12(D1−D2,E1−E2)\overrightarrow{AB} = (\df{D_1 - D_2}2, \df{E_1 - E_2}2) = \df 1 2 (D_1 - D_2, E_1 - E_2)AB=(2D1​−D2​​,2E1​−E2​​)=21​(D1​−D2​,E1​−E2​)

考虑得到的直线方程

(D1−D2)x+(E1−E2)y+F1−F2=0(D_1 - D_2)x + (E_1 - E_2)y + F_1 - F_2 = 0(D1​−D2​)x+(E1​−E2​)y+F1​−F2​=0

其法向量 (D1−D2,E1−E2)(D_1 - D_2, E_1 - E_2)(D1​−D2​,E1​−E2​) 恰好是 2AB→2 \overrightarrow{AB}2AB，平行于 AB→\overrightarrow{AB}AB．

因此 直线与圆心连线垂直．

而先前证明了 直线一定经过两圆交点，因此 直线过切点．

综合两点即得 直线为过切点的公切线，证毕．

注意：上面得到的方程仅能给出 过两圆切点的公切线．两圆内切时，公切线确实仅此一条；而两圆外切时，还有两条 不过两圆切点的公切线，上面的方程无法给出．

切点弦所在直线方程​

对于 ⊙C\odot C⊙C 和在该圆外的一点 PPP，从 PPP 出发可以作 ⊙C\odot C⊙C 的两条切线，从而形成两个切点 AAA 和 BBB．ABABAB 连接得到的弦称作 切点弦，现求解直线 ABABAB 的方程．

一个优于暴力求解，简单的算法如下：

连接 PCPCPC，作 PCPCPC 中点 MMM，不难发现 PPP，AAA，BBB，CCC 都在以 MMM 为圆心，MPMPMP 为半径的 ⊙M\odot M⊙M 上．从而，⊙M\odot M⊙M 和 ⊙C\odot C⊙C 的公共弦就是所求的切点弦．算出公共弦直线方程即可．

上面的 ⊙M\odot M⊙M 其实是以线段 PCPCPC 为直径作出的圆，因此可以利用直径式快速写出 ⊙M\odot M⊙M 的方程．

阿氏圆​

认识阿氏圆​

阿氏圆的内容如下：

对于两个定点 AAA，BBB 和一定值 λ>0\lambda > 0λ>0，若动点 PPP 满足 ∣PA∣∣PB∣=λ\df{|PA|}{|PB|} = \lambda∣PB∣∣PA∣​=λ，则：

  • λ=1\lambda = 1λ=1 时，动点 PPP 的轨迹为 ABABAB 的 垂直平分线．
  • λ≠1\lambda \ne 1λ=1 时，动点 PPP 的轨迹为 一个圆，称作 阿波罗尼斯圆，简称 阿氏圆．
证明

λ=1\lambda = 1λ=1 的情形显然，下讨论 λ≠1\lambda \ne 1λ=1．

不失一般性，设 A(0,0)A(0, 0)A(0,0)，B(m,0)B(m, 0)B(m,0)，动点 P(x,y)P(x, y)P(x,y)，则

x2+y2(x−m)2+y2=λ\df{\sqrt{x^2 + y^2}}{\sqrt{(x - m)^2 + y^2}} = \lambda(x−m)2+y2​x2+y2​​=λ

整理得

(λ2−1)x2+(λ2−1)y2−2λ2mx+λ2m2=0(\lambda^2 - 1)x^2 + (\lambda^2 - 1)y^2 - 2\lambda^2mx + \lambda^2m^2 = 0(λ2−1)x2+(λ2−1)y2−2λ2mx+λ2m2=0

λ>0\lambda > 0λ>0 且 λ≠1\lambda \ne 1λ=1 时，上方程整理得

x2+y2−2λ2mλ2−1x+λ2m2λ2−1=0x^2 + y^2 - \df{2\lambda^2m}{\lambda^2 - 1} x + \df{\lambda^2 m^2}{\lambda^2 - 1} = 0x2+y2−λ2−12λ2m​x+λ2−1λ2m2​=0

已经满足了圆的一般式的结构，下面只需要验证 Δ\DeltaΔ．

Δ=(2λ2mλ2−1)2−4⋅λ2m2λ2−1\Delta = (\df{2\lambda^2m}{\lambda^2 - 1})^2 - 4 \cdot \df{\lambda^2 m^2}{\lambda^2 - 1}Δ=(λ2−12λ2m​)2−4⋅λ2−1λ2m2​(λ2−1)24Δ=(λ2m)2−(λ2m2)(λ2−1)=(λ2m2)λ2−(λ2m2)(λ2−1)=λ2m2>0\bal \df {(\lambda^2 - 1)^2} 4 \Delta &= (\lambda^2m)^2 - (\lambda^2 m^2)(\lambda^2 - 1) \\ &= (\lambda^2m^2)\lambda^2 - (\lambda^2 m^2)(\lambda^2 - 1) \\ &= \lambda^2m^2 > 0 \eal4(λ2−1)2​Δ​=(λ2m)2−(λ2m2)(λ2−1)=(λ2m2)λ2−(λ2m2)(λ2−1)=λ2m2>0​

Δ\DeltaΔ 前的系数 (λ2−1)24\df {(\lambda^2 - 1)^2} 44(λ2−1)2​ 明显恒正，因此 Δ>0\Delta > 0Δ>0，证毕．

阿氏圆的性质​

考察阿氏圆 ⊙M\odot M⊙M 与直线 ABABAB 的交点 CCC 与 DDD，它是 直线 ABABAB 上的两个满足 ∣PA∣∣PB∣=λ\df{|PA|}{|PB|} = \lambda∣PB∣∣PA∣​=λ 的点，且阿氏圆以线段 CDCDCD 为直径．

基于此，求解阿氏圆方程，也可以考虑求出 CCC 与 DDD 的坐标，列直径式求解．但有时 CCC 与 DDD 坐标的形式并不好，如例题 1.3.1，此时笔者觉得朴素求解更好．

对 CCC 与 DDD 作进一步的讨论：

  • 一定有一个为在 线段 ABABAB 上，其将线段 ABABAB 分为 λ:1\lambda : 1λ:1 的两部分，称作 内分点，令其为 CCC．
  • 另一个一定在 线段 ABABAB 外，称作 外分点，令其为 DDD．
    • λ>1\lambda > 1λ>1 时，∣DA∣>∣DB∣|DA| > |DB|∣DA∣>∣DB∣，DDD 在从 BBB 出发远离 AAA 方向的射线上．
    • 0<λ<10 < \lambda < 10<λ<1 时，∣DA∣<∣DB∣|DA| < |DB|∣DA∣<∣DB∣，DDD 在从 AAA 出发远离 BBB 方向的射线上．

综上，线段 ABABAB 与 线段 CDCDCD 一定 有交集，但 不存在包含关系．

换个角度去思考，AAA 与 BBB 中，也一定一个在线段 CDCDCD 上，一个线段 CDCDCD 外．

直线 ABABAB 上的点在线段 CDCDCD 上等价在圆内，因此 AAA 与 BBB 中 一定有一个在阿氏圆内，一定有一个在阿氏圆外．
