量化直线方向的工具

斜率、倾斜角、方向向量之间的关系​

在向量中我们学习过，一个点和一个 非零向量 可以确定一条直线，该向量称作直线的 方向向量．

一条直线的方向向量一定不唯一，设 s\bm ss 是直线 lll 的方向向量，则与 s\bm ss 平行的所有 非零向量 都是 lll 的方向向量，与 s\bm ss 不平行的向量或零向量则均不为 lll 的方向向量．

在学习直线这一节时，我们要有意识地将量化直线方向的工具在一起记忆．上面的方向向量就是我们第一个了解到的量化直线方向的工具，接下来我们再引入两个量化直线方向的工具：斜率和倾斜角．

倾斜角的定义​

对于一条直线 lll，它的 倾斜角 定义如下：

  • 如果 lll 和 xxx 轴相交于一点 AAA，从 AAA 出发，沿着 lll 的向上方向，沿着 xxx 轴正向 分别作出两条射线，这两条射线之间的夹角 α\alphaα 即为 lll 的倾斜角．这种情况下，0<α<π0 < \alpha < \pi0<α<π．
  • 否则，lll 要么和 xxx 轴平行，要么和 xxx 轴重合．此时规定 lll 的倾斜角为 000．

提示：相交只包含两直线交于 恰好一点 的情况，即 重合不是相交．

因此一条直线 lll 的倾斜角 α\alphaα 的取值范围是 [0,π)[0, \pi)[0,π)．

下面给出倾斜角的示例图：

上图中的 α1\alpha_1α1​，α2\alpha_2α2​，α3\alpha_3α3​，α4\alpha_4α4​ 分别是 l1l_1l1​，l2l_2l2​，l3l_3l3​，l4l_4l4​ 的倾斜角．

倾斜角是唯一的

一条直线的倾斜角只有一个，如 α3\alpha_3α3​ 的补角就不是 l3l_3l3​ 的倾斜角．

倾斜角必须是 lll 的 向上方向 和 xxx 轴 正向 形成的夹角．

方向向量和倾斜角的关系​

量化同一方向时，不同的量化方向工具之间应当具有关联．下探究任意直线 lll 的方向向量与倾斜角的关系．

取 lll 的一个 纵坐标非负 的方向向量 s=(x,y)\bm s = (x, y)s=(x,y)．这一定可以做到：任取 lll 的一个方向向量，若其纵坐标为负，取该向量的相反向量即可．

设 xxx 轴正方向单位向量为 i=(1,0)\bm i = (1, 0)i=(1,0)，lll 的倾斜角为 θ\thetaθ，根据倾斜角的定义，有

⟨s,i⟩=θ\la \bm s, \bm i \ra = \theta⟨s,i⟩=θ

根据三角函数的定义，当 x≠0x \ne 0x=0 时，

tan⁡(⟨s,i⟩)=yx\tan(\la \bm s, \bm i \ra ) = \df y xtan(⟨s,i⟩)=xy​

可得

tan⁡θ=yx\tan \theta = \df y xtanθ=xy​

对于 x=0x = 0x=0 的情形，其倾斜角的正切不存在，倾斜角本身等于 π2\df \pi 22π​，为一条 竖直线．

因此，量化同一方向时：

  • 对于 非竖直线，倾斜角的正切等于任一方向向量的纵坐标与横坐标的比．
  • 对于 竖直线，倾斜角的正切不存在，任一方向向量横坐标为 000．

斜率的定义​

在直线倾斜角的正切存在，即直线不竖直时，称 倾斜角的正切 为斜率．斜率常用小写字母 kkk 表示，即

k=tan⁡αk = \tan\alphak=tanα

规定竖直直线斜率不存在．

倾斜角总是存在的

与斜率不同，一条直线的倾斜角总是存在的．

根据斜率的定义，斜率的存在性和倾斜角的正切保持同步，而不和倾斜角保持同步．

斜率与倾斜角的关系是显然的，因此斜率、倾斜角与方向向量三者可互化．

再联系直线任意两点的坐标​

已知直线上不重合的两点的坐标，根据两点确定一条直线，方向也随之确定了．因此，量化方向的工具与直线上任意不重合两点的坐标应当也有联系．

对于直线 lll 上不重合两点 (x1,y1)(x_1, y_1)(x1​,y1​) 和 (x2,y2)(x_2, y_2)(x2​,y2​)，可导出 lll 的一个方向向量 (x,y)=(x1−x2,y1−y2)(x, y) = (x_1 - x_2, y_1 - y_2)(x,y)=(x1​−x2​,y1​−y2​)．这就建立了方向向量与两点坐标的联系．

因此，对于 非竖直线，

k=y1−y2x1−x2=tan⁡α=yxk = \df{y_1 - y_2}{x_1 - x_2} = \tan \alpha = \df y xk=x1​−x2​y1​−y2​​=tanα=xy​

这个公式总结了三个量化直线方向的工具与两点坐标之间的关系，称作 斜率公式．

直线非竖直 是斜率公式的重要使用前提．对于不确定是否竖直的直线，使用斜率公式前，应讨论直线是否竖直（斜率是否存在）．

当然，如果题意确定了直线的斜率存在（如题目条件「直线的斜率是 222」），就不必讨论了．另外，当题目中说「直线的斜率不为 000」这种表述时，也默认题目暗指直线斜率存在．

方向相同的含义​

根据 α∈[0,π)\alpha \in [0, \pi)α∈[0,π) 可知，任意实数 kkk 作为斜率恰好有唯一的倾斜角 α\alphaα 与之对应．

斜率量化直线方向的直观理解​

根据 k=tan⁡αk = \tan \alphak=tanα，令 α\alphaα 从 000 开始逐渐增大，有：

  • α\alphaα 从 000 无限趋近 π2\df \pi 22π​ 的过程中，kkk 从 000 增大，无限趋近 +∞+\infty+∞，增长速度越来越快．
  • α\alphaα 跨越 π2\df \pi 22π​ 的时候，kkk 从 +∞+\infty+∞ 突变为 −∞-\infty−∞．
  • α\alphaα 从 π2\df \pi 22π​ 到接近 π\piπ 的过程中，kkk 从 −∞-\infty−∞ 增大，趋近于 000，增长速度越来越慢．

另外，∣k∣|k|∣k∣ 越大，直线的方向和 yyy 轴越接近，我们认为这个直线越「陡峭」．

斜率量化直线方向的优势在于：它是一种可以用全体实数量化绝大部分直线方向的工具（只有竖直方向无法量化，当然读者也可以认为斜率不存在也是一种变相量化竖直方向的方式）．

过定点直线交于线段求斜率范围​

例题 2.5.1

已知 A(−2,3)A(-2, 3)A(−2,3)，B(1,1)B(1, 1)B(1,1)，OOO 为坐标系原点．过 OOO 的直线 lll 与线段 ABABAB 相交，求 lll 的斜率取值范围．

这种题目步骤比较固定，如下：

  • 下设直线必经点为 PPP（例题 2.5.1 中这个 PPP 就是坐标系原点 OOO），线段的两个端点分别为 AAA 和 BBB．
  • 分别作出 PAPAPA，PBPBPB 两条直线，并算出两条直线的斜率．
  • 判断直线能取的范围是从 PAPAPA 逆时针扫到 PBPBPB 这个范围，还是能从 PBPBPB 逆时针扫到 PAPAPA 这个范围（钦定 逆时针 旋转是因为逆时针旋转时 α\alphaα 增大，斜率也在分段增大，不易出错）．

旋转过程中，有两个分段点需要注意：

  • 直线 逆时针 扫过与 yyy 轴平行的线（称之为「竖直线」）时，斜率 从 +∞+\infty+∞ 突变为 −∞-\infty−∞，倾斜角不突变．
  • 直线 逆时针 扫过与 xxx 轴平行的线（称之为「水平线」）时，倾斜角 从趋近 π\piπ 突变为 000，斜率不突变．

如果求斜率范围，注意「竖直线」为分段点；如果求倾斜角范围，注意「水平线」为分段点．

根据题目画图．

作出直线 OAOAOA，OBOBOB．不难发现 lll 能扫过的范围是从 OBOBOB 逆时针扫到 OAOAOA 这一段，考虑其斜率变化．

隐斜率​

指的是形如 y1−y2x1−x2\df{y_1 - y_2}{x_1 - x_2}x1​−x2​y1​−y2​​ 的分式可以看成是 (x1,y1)(x_1, y_1)(x1​,y1​) 和 (x2,y2)(x_2, y_2)(x2​,y2​) 连线的斜率，从而利用数形结合解决问题．

直线的位置关系​

刻画几何条件的意义​

「刻画几何条件」，就是将一个陌生的几何条件 PPP 等价 为一个容易处理的几何条件 QQQ．

这么做的目的是：

  • 当题目的条件中出现几何条件 PPP 时，我们可以把这个条件直接等价为容易处理的几何条件 QQQ，从而解决题目．
  • 当题目的结论是几何条件 PPP 时，我们可以转而证明几何条件 QQQ．

举例说明．比如后面会学习到「AAA，BBB，CCC 三点共线」等价为「ABABAB 和 ACACAC 方向相同」．这就是一个几何条件的刻画方式．那么在题目中：

  • 如果题目的条件是「AAA，BBB，CCC 三点共线」，我们可以把这个条件等价为「ABABAB 和 ACACAC 方向相同」，再做题．
  • 如果题目要求证明「AAA，BBB，CCC 三点共线」．我们可以转而去证明「ABABAB 和 ACACAC 方向相同」．

不同的刻画方式可能具有不同的解题效果．在后面的学习中，将会介绍很多常见的几何条件的通常最合适的刻画方式，便于读者做题的效率和正确率兼备．

平行、相交、垂直​

重合和三点共线​

等价刻画「l1l_1l1​ 和 l2l_2l2​ 重合」的两种方式：

  • l1l_1l1​ 和 l2l_2l2​ 方向相同 并且 经过同一个点（一个点和一个方向确定一条直线）．
  • l1l_1l1​ 的方程和 l2l_2l2​ 的 方程等价．

一种等价刻画「AAA，BBB，CCC 三点共线」的方式是：直线 ABABAB 和直线 ACACAC 的方向相同．即 从三点中的一点出发，向另外两点连线，连线方向相同与三点共线等价．

证明这个刻画的等价性．如果直线 ABABAB 和直线 ACACAC 方向相同，则又因为它们经过同一点 BBB，直线 ABABAB 和 ACACAC 重合．那么点 BBB 在直线 ABABAB 上的同时，也一定在直线 ACACAC 上．同时，点 AAA 和点 CCC 也在直线 ACACAC 上，因此 AAA，BBB，CCC 三点共线．

而如果 AAA，BBB，CCC 三点共线，显然直线 ABABAB 和直线 ACACAC 的方向相同．所以上面的刻画等价．

不重合​

先前讲到，两直线平行的一种等价刻画方式为：两直线方向相同且证明两直线不重合．两直线不重合如何进一步刻画？

要证明 l1l_1l1​ 和 l2l_2l2​ 不重合：

  • 当方程可知，可以证明 l1l_1l1​ 和 l2l_2l2​ 方程不等价．
  • 否则，可以在两直线上各取一点连线，证明该线的方向不同于两直线中的某一条．

夹角、到角和倾斜角、斜率的关系​

两条直线方向向量和夹角的关系，可直接使用向量夹角的知识求解．下面讨论夹角和倾斜角与斜率的关系，并引入一个新概念：到角．

两相交直线的 夹角（两直线所成角） 的定义为：两直线所成的两对对顶角中，较小 的一对角中，其中一个的大小．

一条直线 l1l_1l1​ 到另一条相交直线 l2l_2l2​ 的 到角 的定义为：l1l_1l1​ 沿着与 l2l_2l2​ 的唯一交点 逆时针旋转 到第一次与 l2l_2l2​ 重合时的旋转角．将 l1l_1l1​ 称作到角的 始边，l2l_2l2​ 称作到角的 终边．

对于平行或重合的直线，认为它们不存在夹角和到角．

  • l1l_1l1​ 到 l2l_2l2​ 的到角为 锐角或直角 时，它就是 l1l_1l1​ 和 l2l_2l2​ 的夹角．
  • l1l_1l1​ 到 l2l_2l2​ 的到角为 钝角 时，它是 l1l_1l1​ 和 l2l_2l2​ 夹角的 补角．
  • 到角的范围是 (0,π)(0, \pi)(0,π)，而夹角的范围是 (0,π2](0, \df \pi 2](0,2π​]．

对于任意一个角 α\alphaα，{β∈[0,π)∣β+kπ,k∈Z}\{\beta \in [0, \pi) \mid \beta + k\pi, k \in \Z\}{β∈[0,π)∣β+kπ,k∈Z} 恰有唯一元素 β\betaβ，记 β=α mod π\beta = \alpha \bmod \piβ=αmodπ．

两直线 到角 与 倾斜角 的关系：l1l_1l1​ 倾斜角为 α\alphaα，l1l_1l1​ 到 l2l_2l2​ 的到角为 θ\thetaθ，则 l2l_2l2​ 倾斜角为 (α+θ) mod π(\alpha + \theta) \bmod \pi(α+θ)modπ．

两直线 夹角 与 倾斜角 的关系：l1l_1l1​ 倾斜角为 α\alphaα，l1l_1l1​ 与 l2l_2l2​ 的夹角为 θ\thetaθ，则 l2l_2l2​ 的倾斜角为 (α+θ) mod π(\alpha + \theta) \bmod \pi(α+θ)modπ 或 (α−θ) mod π(\alpha - \theta) \bmod \pi(α−θ)modπ，对应 l1l_1l1​ 到 l2l_2l2​ 的到角分别为 θ\thetaθ 和 π−θ\pi - \thetaπ−θ．

上面是两直线到角和夹角和两直线倾斜角之间的关系，现试探究两直线到角和夹角和两直线斜率之间的关系．

当 l1l_1l1​ 到 l2l_2l2​ 的到角为 θ\thetaθ 时，我们有 θ=(α2−α1) mod π\theta = (\alpha_2 - \alpha_1) \bmod \piθ=(α2​−α1​)modπ．当 θ≠π2\theta \ne \df \pi 2θ=2π​ 且 α1≠π2\alpha_1 \ne \df \pi 2α1​=2π​ 且 α2≠π2\alpha_2 \ne \df \pi 2α2​=2π​ 时，有：

tan⁡θ=tan⁡[(α2−α1) mod π]=tan⁡(α2−α1)=tan⁡α2−tan⁡α11+tan⁡α1tan⁡α2=k2−k11+k1k2\tan \theta = \tan[(\alpha_2 - \alpha_1) \bmod \pi] = \tan(\alpha_2 - \alpha_1) = \df{\tan \alpha_2 - \tan \alpha_1}{1 + \tan\alpha_1\tan\alpha_2} = \df{k_2 - k_1}{1 + k_1k_2}tanθ=tan[(α2​−α1​)modπ]=tan(α2​−α1​)=1+tanα1​tanα2​tanα2​−tanα1​​=1+k1​k2​k2​−k1​​

也即 l1l_1l1​ 到 l2l_2l2​ 的到角为 θ\thetaθ 时，tan⁡θ=k2−k11+k1k2\tan\theta = \df{k_2 - k_1}{1 + k_1k_2}tanθ=1+k1​k2​k2​−k1​​．这条结论称作 到角公式，它给出了一直线到另一直线的到角和两直线斜率的关系．注意分子上是 终边的斜率减去始边的斜率，不能反．

当 l1l_1l1​ 和 l2l_2l2​ 的夹角为 θ\thetaθ 时，l1l_1l1​ 到 l2l_2l2​ 的到角为 θ\thetaθ 或 π−θ\pi - \thetaπ−θ．当 θ≠π2\theta \ne \df \pi 2θ=2π​ 且 α1≠π2\alpha_1 \ne \df \pi 2α1​=2π​ 且 α2≠π2\alpha_2 \ne \df \pi 2α2​=2π​ 时，我们有 θ∈(0,π2)\theta \in (0, \df \pi 2)θ∈(0,2π​)，此时 tan⁡θ>0\tan \theta > 0tanθ>0，tan⁡(π−θ)<0\tan(\pi - \theta) < 0tan(π−θ)<0．

  • 当 l1l_1l1​ 到 l2l_2l2​ 的到角为 θ\thetaθ 时，有 tan⁡θ=∣tan⁡θ∣=∣k1−k21+k1k2∣\tan \theta = \lv \tan \theta\rv = \lv \df{k_1 - k_2}{1 + k_1k_2}\rvtanθ=∣tanθ∣=​1+k1​k2​k1​−k2​​​．
  • 当 l1l_1l1​ 到 l2l_2l2​ 的到角为 π−θ\pi - \thetaπ−θ 时，有 tan⁡θ=−tan⁡(π−θ)=∣tan⁡(π−θ)∣=∣k1−k21+k1k2∣\tan \theta = -\tan(\pi - \theta) = \lv \tan (\pi - \theta)\rv = \lv \df{k_1 - k_2}{1 + k_1k_2} \rvtanθ=−tan(π−θ)=∣tan(π−θ)∣=​1+k1​k2​k1​−k2​​​．

综上 l1l_1l1​ 和 l2l_2l2​ 的夹角为 θ\thetaθ 时，tan⁡θ=∣k1−k21+k1k2∣\tan \theta = \lv\df{k_1 - k_2}{1 + k_1k_2}\rvtanθ=​1+k1​k2​k1​−k2​​​．这条结论称作 夹角公式，也即到角公式上套一个绝对值，这时分子相减的顺序不重要．

需要注意的是，到角公式和夹角公式并不是课本上公式，不要在大题中使用．

到角公式和夹角公式的使用前提：两直线斜率均存在且夹角不为直角．

直线中的位置关系题型​

统一斜率原则​

给定 l1l_1l1​，l2l_2l2​ 的方向信息（为方向向量、倾斜角、斜率、两点坐标中的一种），求 l1l_1l1​ 与 l2l_2l2​ 的位置关系．

  • 如果两直线给出的方向信息为 同一种（如均为倾斜角），直接套用上面讲过的结论即可．
  • 如果 不是同一种，将两直线的方向信息都 统一成斜率 再做．这就是统一斜率原则．

为什么要选择统一成斜率？因为无论是已知了直线的方向向量，还是倾斜角，还是不重合的两点坐标，算出斜率都是很简单的．如果斜率存在，方向向量直接将坐标比一下即可，倾斜角直接正切一下即可，不重合两点坐标直接代入斜率公式即可．

而倾斜角无论是从斜率，从两点坐标，还是从方向向量，都需要一个「逆正切函数」的过程；而方向向量从倾斜角得到也必须先转成斜率．综上，统一成斜率一般是最简单的．

另外如果两直线给出的方向信息为同一种，但都是两点坐标时，也要统一成斜率做．原因很简单，并没有「直线上两点坐标」和「直线位置关系」之间关系的直接结论．

需要特别注意 平行的验重 和 斜率不存在 两种容易遗漏的情况．

已知直线位置关系求参数​

已知直线位置关系求参数，可以沿用上面的策略：将两直线的方向统一成斜率表示．

具体来说，我们可以将两直线的斜率表示成带参数的式子，然后再根据已知的直线位置关系列出两个斜率应该满足的式子，从而解出参数 kkk．

注意事项还是平行的验重和斜率公式的使用前提，判断斜率是否存在．

例题 4.2.1

已知过 A(m,1)A(m, 1)A(m,1)，B(−1,m)B(-1, m)B(−1,m) 两点的直线 l1l_1l1​ 和过 P(1,2)P(1, 2)P(1,2)，Q(−5,0)Q(-5, 0)Q(−5,0) 两点的直线 l2l_2l2​ 平行，求 mmm 的值．

例题 4.2.1 解答

l2l_2l2​ 存在斜率 k2=2−01−(−5)=13k_2 = \df{2 - 0}{1 - (-5)} = \df 1 3k2​=1−(−5)2−0​=31​．因此，l1l_1l1​ 的斜率必须存在且等于 l2l_2l2​ 的斜率，即 k1=13k_1 = \df 1 3k1​=31​．

mmm 取什么值都不能使 AAA，BBB 两点重合．根据斜率公式，有 1−mm−(−1)=13\df{1 - m}{m - (-1)} = \df 1 3m−(−1)1−m​=31​，解得 m=12m = \df 1 2m=21​．

现验证 m=12m = \df 1 2m=21​ 时 l1l_1l1​ 和 l2l_2l2​ 不重合．因为 kAP=2−11−12=2≠k2k_{AP} = \df{2 - 1}{1 - \df 1 2} = 2 \ne k_2kAP​=1−21​2−1​=2=k2​，所以 l1l_1l1​ 和 l2l_2l2​ 不重合．

例题 4.2.2

已知过 A(m,1)A(m, 1)A(m,1)，B(−1,m)B(-1, m)B(−1,m) 两点的直线 l1l_1l1​ 和过 P(1,2)P(1, 2)P(1,2)，Q(−5,0)Q(-5, 0)Q(−5,0) 两点的直线 l2l_2l2​ 垂直，求 mmm 的值．

例题 4.2.2 解答

l2l_2l2​ 存在斜率 k2=2−01−(−5)=13k_2 = \df{2 - 0}{1 - (-5)} = \df 1 3k2​=1−(−5)2−0​=31​．因此，l1⊥l2l_1 \perp l_2l1​⊥l2​ 等价于 l1l_1l1​ 的斜率存在且 k1k2=−1k_1k_2 = -1k1​k2​=−1．

mmm 取什么值都不能使 AAA，BBB 两点重合．根据斜率公式，有 k1=1−mm−(−1)k_1 = \df{1 - m}{m - (-1)}k1​=m−(−1)1−m​，于是有 1−mm−(−1)⋅13=−1\df{1 - m}{m - (-1)} \cdot \df 1 3 = -1m−(−1)1−m​⋅31​=−1，解得 m=−2m = -2m=−2．

直线呈一定角度​

例题 4.3

若正方形一条对角线所在直线斜率为 222，求该正方形两条邻边所在直线的斜率分别是多少．（填空题）

例题 4.3 不用夹角公式的解答

正方形两条邻边分别和对角线呈 π4\df \pi 44π​ 的夹角．设对角线所在直线倾斜角为 α\alphaα，根据 tan⁡α=2\tan \alpha = 2tanα=2，不难发现有 α∈(π4,π2)\alpha \in (\df \pi 4, \df \pi 2)α∈(4π​,2π​)，因此两邻边所在直线的倾斜角分别为 α−π4\alpha - \df \pi 4α−4π​ 和 α+π4\alpha + \df \pi 4α+4π​．

现在要求两邻边所在直线的斜率，即 tan⁡(α−π4)\tan(\alpha - \df \pi 4)tan(α−4π​) 和 tan⁡(α+π4)\tan(\alpha + \df \pi 4)tan(α+4π​)．以计算 tan⁡(α+π4)\tan(\alpha + \df \pi 4)tan(α+4π​) 为例，用正切和角展开为 tan⁡α+tan⁡π41−tan⁡αtan⁡π4\df{\tan\alpha + \tan \fr \pi 4}{1 - \tan\alpha\tan\fr \pi 4}1−tanαtan4π​tanα+tan4π​​，将 tan⁡α=2\tan \alpha = 2tanα=2 代入可得结果为 −3-3−3．同理可得另外一个结果为 13\df 1 331​．

计算可以更快：注意到正方形两邻边垂直，其所在直线斜率互为负倒数．所以将一个直线的斜率算出来后，另一个直线斜率采负倒数即可．

例题 4.3 采用夹角公式的解答

显然所求的两直线以及对角线所在直线斜率存在，且对角线和所求直线夹角不为直角，可以使用夹角公式．

设所求直线斜率为 kkk，则 kkk 应当满足 ∣2−k1+2k∣=1\lv \df{2 - k}{1 + 2k} \rv = 1​1+2k2−k​​=1，两解分别为 −3-3−3 和 13\df 1 331​，就是所求的两直线斜率．
