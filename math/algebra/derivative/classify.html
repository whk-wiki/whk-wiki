讨论含参函数的单调性

对于函数的最值、零点、不等式等问题，

当函数 f(x)f(x)f(x) 含参数 mmm 时，其单调性分布与 mmm 有关，对于不同的 mmm，函数 f(x)f(x)f(x) 的单调性不同，而对函数的最值、零点、不等式等研究的基础都是单调性，因此 讨论含参函数的单调性 是一个 非常重要，也非常基础 的技能．读者应该在这类题目达到将近百分百的正确率．

我们知道，函数 f(x)f(x)f(x) 的 单调性分布 与 f′(x)f'(x)f′(x) 的 正负分布 存在对应，因此讨论含参函数的单调性，就是在讨论这个含参函数的导数（一般也含参）的 正负分布．这个操作与我们在不等式一章学习过的 讨论不等式的解集 有相似之处，但会更加有难度，并且部分细节存在差异，所以这里会重新系统梳理．

讨论分类基本原则​

对函数 f(x)f(x)f(x) 的单调性讨论，就是对导数的 f′(x)f'(x)f′(x) 的正负分布进行讨论．分类基本原则有四条，请记在笔记上：

  • f′(x)f'(x)f′(x) 在 自然定义域 上的 单调性分布．
  • f′(x)f'(x)f′(x) 在 自然定义域 上的 零点数量．
  • f′(x)f'(x)f′(x) 在自然定义域上的 每个零点，与 约束定义域 的 位置关系．
  • f′(x)f'(x)f′(x) 在 约束定义域 内的 零点 间 大小关系．

下面明确 自然定义域 与 约束定义域 的定义．如

f(x)=mx+ln⁡xf(x) = mx + \ln xf(x)=mx+lnx

其导数为

f′(x)=m+1x,x>0f'(x) = m + \df 1 x, \quad x > 0f′(x)=m+x1​,x>0

这里称导数的 自然定义域 是 {x∣x≠0}\{x \mid x \ne 0\}{x∣x=0}，约束定义域 是 (0,+∞)(0, +\infty)(0,+∞)．即，

  • 自然定义域 就是从 导数的表达式直接呈现 出的定义域，是 直接方便我们研究导函数性质 的定义域．
  • 而 约束定义域 是因原函数定义域的限制，比自然定义域严格 的，实际上 真正正确 的定义域．
  • 约束定义域是自然定义域的子集．

导数为单调函数​

如果待考察正负分布的含参导数 f′(x)f'(x)f′(x) 满足特征：无论参数取何值，f′(x)f'(x)f′(x) 在自然定义域具有单调性，则属于此类情况．例如：

  • 含参一次导数：如 f′(x)=(m+3)x+mf'(x) = (m + 3)x + mf′(x)=(m+3)x+m．
  • 含参指对导数：如 f′(x)=ln⁡x+mf'(x) = \ln x + mf′(x)=lnx+m，f′(x)=ex+mf'(x) = \e^x + mf′(x)=ex+m，f′(x)=mex+2f'(x) = m\e^x + 2f′(x)=mex+2．

这类导数由于在自然定义域上有单调性，因此在自然定义域上 最多只有一个零点．根据我们的四条分类原则，这类导数的讨论分类原则如下：

  • 导数的 单调性 是递减 / 递增 / 常函数？
  • 导数在 自然定义域 上 有无零点？
  • 如果导数在自然定义域上存在零点，该零点在定义域的左侧 / 中间 / 右侧？

由于这类导数零点最多只有一个，零点大小关系无需讨论．
