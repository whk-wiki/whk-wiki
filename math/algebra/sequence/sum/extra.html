不常规裂项​

裂项的本质​

上一页讲解的的 常规裂项（也是最常考的裂项方式），裂项结果为

C⋅(1bn−1bn+k),C为与n无关的常数C \cdot (\df 1 {b_n} - \df 1 {b_{n + k}}), \quad C 为与 n 无关的常数C⋅(bn​1​−bn+k​1​),C为与n无关的常数

其求和时：

  • CCC 作为公因式提出．
  • 中间括号的求和，整体构成了数列 {1bn}\{\df 1 {b_n}\}{bn​1​} 的两个连续段相减．
  • 运用 nnn，kkk 互换将不定长 nnn 的两段连续段相减变为定长 kkk 的两段连续段相减．

那么，为什么非得是 {1bn}\{\df 1 {b_n}\}{bn​1​} 呢？只要是一个数列就可以了！因此，更一般的裂项应当总结为

C⋅(bn−bn+k),C为与n无关的常数C \cdot (b_n - b_{n + k}), \quad C 为与 n 无关的常数C⋅(bn​−bn+k​),C为与n无关的常数

这里的 bnb_nbn​ 可以像 常规裂项 一样是 1qn\df 1 {q_n}qn​1​ 的形式，当然也可以是 pnqn\df{p_n}{q_n}qn​pn​​ 的形式，当然也可以不是一个分式，比如 pn\sqrt{p_n}pn​​，pn2{p_n}^2pn​2……都可以．

因此，裂项的内容绝对不止于此，我们可以进行一些更为深入的研究．

不常规裂项一 - 分子非常数​

C⋅(cnbn−cn+kbn+k),C为与n无关的常数C \cdot (\df {c_n}{b_n} - \df {c_{n + k}}{b_{n + k}}), \quad C 为与 n 无关的常数C⋅(bn​cn​​−bn+k​cn+k​​),C为与n无关的常数

这个裂项求和的结果最后也能表示成数列 {cnbn}\{\df{c_n}{b_n}\}{bn​cn​​} 的 两个连续段相减 形式，因此也是一个裂项，后面称这种裂项为 分子非常数裂项．

分子非常规裂项 的步骤并不固定，通常需要 先猜后证，以例题为例说明．

例题 1.1.1

求 an=n⋅2n(n+1)(n+2)a_n = \df{n \cdot 2^n}{(n + 1)(n + 2)}an​=(n+1)(n+2)n⋅2n​ 的前缀和 SnS_nSn​．

本题无法用常规裂项策略解决，读者可以自行试试看：

  • 分母可以表示成同一个数列 {n+1}\{n + 1\}{n+1} 的第 nnn 项和第 n+1n + 1n+1 项．
  • 但 分子分母差\df{分子}{分母差}分母差分子​ 不是常数．

并且似乎无论如何变形都无法常规裂项，遂放弃，尝试分子非常数裂项．

例题 1.1 解答（思考过程）

分子非常数裂项中，可以猜测一下它的裂项结果．这里笔者会猜测

n⋅2n(n+1)(n+2)=?2nn+1−2n+1n+2\df{n \cdot 2^n}{(n + 1)(n + 2)} \stackrel ? = \df{2^n}{n + 1} - \df{2^{n + 1}}{n + 2}(n+1)(n+2)n⋅2n​=?n+12n​−n+22n+1​

为什么会想到这么猜？首先：

  • 猜出来的形式一定要是一个合法裂项，即分母可以统一成一个数列 {bn}\{b_n\}{bn​} 的第 nnn 项与第 n+kn + kn+k 项，分子也能统一成一个数列 {cn}\{c_n\}{cn​} 的第 nnn 项与第 n+kn + kn+k 项．两个 kkk 是相同的（这里是 111）．
  • 优先猜测结构与通项相似的裂项．这里分母全是一次式，分子含 222 的幂，因此笔者会这么猜．

然后再尝试证明：

2nn+1−2n+1n+2=2n⋅(n+2)−2n+1⋅(n+1)(n+1)(n+2)=−n⋅2n(n+1)(n+2)\df{2^n}{n + 1} - \df{2^{n + 1}}{n + 2} = \df{2^n\cdot (n + 2) - 2^{n + 1}\cdot (n + 1)}{(n + 1)(n + 2)} = \df{-n \cdot 2^n}{(n + 1)(n + 2)}n+12n​−n+22n+1​=(n+1)(n+2)2n⋅(n+2)−2n+1⋅(n+1)​=(n+1)(n+2)−n⋅2n​

结果最终的结果与原式呈相反数，说明我们要提一个 −1-1−1 的公因式出来：

n⋅2n(n+1)(n+2)=−(2nn+1−2n+1n+2)\df{n \cdot 2^n}{(n + 1)(n + 2)} = -(\df{2^n}{n + 1} - \df{2^{n + 1}}{n + 2})(n+1)(n+2)n⋅2n​=−(n+12n​−n+22n+1​)

裂项成功，后面就轻车熟路了．

例题 1.1 解答（写在卷子上的过程）

注意到 an=−(2nn+1−2n+1n+2)a_n = -(\df{2^n}{n + 1} - \df{2^{n + 1}}{n + 2})an​=−(n+12n​−n+22n+1​)，因此有

Sn=−(22−223+223−234+⋯+2nn+1−2n+1n+2)=−[(22+223+⋯+2nn+1)−(223+234+⋯+2n+1n+2)]=−(1−2n+1n+2)=2n+1n+2−1\begin{aligned} S_n &= -(\df 2 2 - \df{2^2}3 + \df {2^2}{3} - \df {2^3}4 + \cdots + \df{2^n}{n + 1} - \df{2^{n + 1}}{n + 2}) \\ &= -[(\df 2 2 + \df {2^2} 3 + \cdots + \df{2^n}{n + 1}) - (\df {2^2} 3 + \df {2^3} 4 + \cdots + \df{2^{n + 1}}{n + 2}) ] \\ &= -(1 - \df{2^{n + 1}}{n + 2}) \\ &= \df{2^{n + 1}}{n + 2} - 1 \end{aligned}Sn​​=−(22​−322​+322​−423​+⋯+n+12n​−n+22n+1​)=−[(22​+322​+⋯+n+12n​)−(322​+423​+⋯+n+22n+1​)]=−(1−n+22n+1​)=n+22n+1​−1​

一般来说，分子非常数裂项的试题，则裂项结果通常是较好猜的．有可能出现需要系数修正的情况，如上题．

例题 1.1.2

求 an=(n−1)(n+2)2na_n= \df{(n - 1)(n + 2)}{2^n}an​=2n(n−1)(n+2)​ 的前缀和 SnS_nSn​．

例题 1.1.2 解答

猜测与通项结构相同的裂项：

(n−1)(n+2)2n=?(n+c1)(n+c2)2n−(n+c1+1)(n+c2+1)2n+1\df{(n - 1)(n + 2)}{2^n} \stackrel ? = \df{(n + c_1)(n + c_2)}{2^n} - \df{(n + c_1 + 1)(n + c_2 + 1)}{2^{n + 1}}2n(n−1)(n+2)​=?2n(n+c1​)(n+c2​)​−2n+1(n+c1​+1)(n+c2​+1)​

首先可以注意到

2(n+c1)(n+c2)−(n+c1+1)(n+c2+1)=2(n−1)(n+2)2(n + c_1)(n + c_2) - (n + c_1 + 1)(n + c_2 + 1) = 2(n - 1)(n + 2)2(n+c1​)(n+c2​)−(n+c1​+1)(n+c2​+1)=2(n−1)(n+2)

会发现不太对劲，左侧 n2n^2n2 的系数是 111，右侧是 222，对不上．

因此修正猜测

(n−1)(n+2)2n=?(n+c1)(n+c2)2n−1−(n+c1+1)(n+c2+1)2n\df{(n - 1)(n + 2)}{2^n} \stackrel ? = \df{(n + c_1)(n + c_2)}{2^{n - 1}} - \df{(n + c_1 + 1)(n + c_2 + 1)}{2^n}2n(n−1)(n+2)​=?2n−1(n+c1​)(n+c2​)​−2n(n+c1​+1)(n+c2​+1)​

此时

2(n+c1)(n+c2)−(n+c1+1)(n+c2+1)=(n−1)(n+2)=n2+n−22(n + c_1)(n + c_2) - (n + c_1 + 1)(n + c_2 + 1) = (n - 1)(n + 2) = n^2 + n - 22(n+c1​)(n+c2​)−(n+c1​+1)(n+c2​+1)=(n−1)(n+2)=n2+n−2

左侧一次项系数为 2c1+2c2−c1−1−c2−1=c1+c2−22c_1 + 2c_2 - c_1 - 1 - c_2 - 1 = c_1 + c_2 - 22c1​+2c2​−c1​−1−c2​−1=c1​+c2​−2，应为 111，可知 c1+c2=3c_1 + c_2 = 3c1​+c2​=3．

左侧常数项为 2c1c2−(c1+1)(c2+1)=c1c2−c1−c2−12c_1c_2 - (c_1 + 1)(c_2 + 1) = c_1c_2 - c_1 - c_2 - 12c1​c2​−(c1​+1)(c2​+1)=c1​c2​−c1​−c2​−1，应为 −2-2−2，可知 c1c2=2c_1c_2 = 2c1​c2​=2．

很明显 c1=1c_1 = 1c1​=1，c2=2c_2 = 2c2​=2，这就边猜边证出来了

(n−1)(n+2)2n=(n+1)(n+2)2n−(n+2)(n+3)2n+1\df{(n - 1)(n + 2)}{2^n} = \df{(n + 1)(n + 2)}{2^n} - \df{(n + 2)(n + 3)}{2^{n + 1}}2n(n−1)(n+2)​=2n(n+1)(n+2)​−2n+1(n+2)(n+3)​

一个很有趣的数学巧合是上面的式子中 n+2n + 2n+2 可以约去，但约去后就不再是裂项了，读者可以试试看．

例题 1.3

例题 1.3 解答
4n2(2n+1)(2n−1)=4n24n2−1=14n2−1+1=12(12n−1−12n+1)+1\df{4n^2}{(2n + 1)(2n - 1)} = \df{4n^2}{4n^2 - 1} = \df{1}{4n^2 - 1} + 1 = \df 1 2 (\df 1 {2n - 1} - \df 1 {2n + 1}) \* + 1(2n+1)(2n−1)4n2​=4n2−14n2​=4n2−11​+1=21​(2n−11​−2n+11​)+1

注意到这里带了个 +1+1+1 的常数小尾巴，但它并不影响这个形式可以求和：

  • 每一项有个 +1+ 1+1，整体对 SnS_nSn​ 就有 +n+n+n 的贡献，单独计算它．
  • 而剩余的裂项部分求和．

最后再将两部分计算总和即可．

这个例子提示我们：如果待裂项分式 可分离常数，可以 优先考虑分离常数．下面再给一个分离常数的例子：

an=n2+n+1n2+n  ⟹  an=1+1n2+n=1+1n−1n+1a_n = \df{n^2 + n + 1}{n^2 + n} \implies a_n = 1 + \df 1 {n^2 + n} = 1 + \df 1 n - \df 1 {n + 1}an​=n2+nn2+n+1​⟹an​=1+n2+n1​=1+n1​−n+11​

这个尾巴可以是常数 +1+1+1，还可以是一个能求和的式子，如 +n+ n+n（+n+n+n 的贡献可以单独求和为 n(n+1)2\df{n(n + 1)}22n(n+1)​），但这种情况一般少见．

因此裂项被我们拓展为

C⋅(bn−bn+k)+cnC \cdot (b_n - b_{n + k}) + c_nC⋅(bn​−bn+k​)+cn​
  • CCC 为与 nnn 无关的常数．
  • {cn}\{c_n\}{cn​} 为可求前缀和的数列（常数视作常数列）．
