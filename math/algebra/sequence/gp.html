等比数列

等比数列基础​

等比数列基本概念​

如果一个数列从第二项起，每一项与前一项的比都为一个常数 qqq（quotient），则将这个数列称作 等比数列，qqq 称作该数列的 公比．

根据定义可以得知：

  • 一个等比数列中，每一项都不为 000（否则做比操作不合法）．
  • q≠0q \ne 0q=0（因为任意一项均不为 000）．
例题 1.1

设等比数列 {an}\{a_n\}{an​} 公比为 qqq，下列说法正确的是（ ）．

  • A. {anan+1}\{a_na_{n + 1}\}{an​an+1​} 是公比为 qqq 的等比数列．
  • B. {an+an+1}\{a_n + a_{n + 1}\}{an​+an+1​} 是公比为 qqq 的等比数列．
  • C. {an−an+1}\{a_n - a_{n + 1}\}{an​−an+1​} 是公比为 qqq 的等比数列．
  • D. {1an}\{ \dfrac 1 {a_n} \}{an​1​} 是公比为 1q\dfrac 1 qq1​ 的等比数列．
例题 1.1 解答

此类题目应先看结果数列是否可能有 000，有 000 的一定不是等比数列．

对 B 可以构造 1,−1,1,−1,…1, -1, 1, -1, \ldots1,−1,1,−1,…，C 可以构造 1,1,1,…1, 1, 1, \ldots1,1,1,…，此时对应数列会出现 000，因此不选．

等比数列任一项不为 000，因此 anan+1≠0a_na_{n + 1} \ne 0an​an+1​=0．然而 an+1an+2anan+1=q2\dfrac{a_{n + 1}a_{n + 2}}{a_na_{n + 1}} = q^2an​an+1​an+1​an+2​​=q2，不一定等于 qqq，因此 A 错误．

而 1an+1÷1an=anan+1=1q\dfrac 1 {a_{n + 1}} \div \dfrac 1 {a_n} = \dfrac {a_n}{a_{n + 1}} = \dfrac 1 qan+1​1​÷an​1​=an+1​an​​=q1​，D 正确．

对于等比数列 {an}\{a_n\}{an​}，设其「比分」为 {Rn}\{R_n\}{Rn​}，有

Rn={a1,n=1q,n≥2R_n = \begin{cases} a_1, & n = 1 \\ q, & n \ge 2 \end{cases}Rn​={a1​,q,​n=1n≥2​

对 {Rn}\{R_n\}{Rn​} 做前缀积可得

an=a1qn−1a_n = a_1 q^{n - 1}an​=a1​qn−1

该式为 等比数列的通项公式．确定 a1a_1a1​ 和 qqq 后，等比数列确定，而只确定其中之一则无法确定等比数列．

对于长度为 333 的等比数列 a,b,ca, b, ca,b,c，称 {bn}\{b_n\}{bn​} 是 {an}\{a_n\}{an​} 和 {cn}\{c_n\}{cn​} 的 等比中项．这等价于

ba=cb\dfrac b a = \dfrac c bab​=bc​

又等价于

{b2=acabc≠0\begin{cases} b^2 = ac \\ abc \ne 0 \end{cases}{b2=acabc=0​

单向推出而不等价于 b2=acb^2 = acb2=ac．

也就是说，{bn}\{b_n\}{bn​} 是 {an}\{a_n\}{an​} 和 {cn}\{c_n\}{cn​} 的等比中项可以推出 b2=acb^2 = acb2=ac，但反过来，知道 b2=acb^2 = acb2=ac 还不足以推出 {bn}\{b_n\}{bn​} 是 {an}\{a_n\}{an​} 和 {cn}\{c_n\}{cn​} 的等比中项，必须再加条件 abc≠0abc \ne 0abc=0．这是与等差中项不同的地方．

还有一个明显不同的地方是：对于任意 {an}\{a_n\}{an​} 和 {cn}\{c_n\}{cn​}，其一定恰存在一个等差中项 {bn}\{b_n\}{bn​}；但对于等比数列：

  • 如果 ac>0ac > 0ac>0，{an}\{a_n\}{an​} 与 {cn}\{c_n\}{cn​} 存在两个等比中项 b=±acb = \pm \sqrt{ac}b=±ac​．
  • 如果 ac<0ac < 0ac<0，{an}\{a_n\}{an​} 与 {cn}\{c_n\}{cn​} 不存在等比中项．

如 111 和 999 的等比中项为 ±3\pm 3±3，333 和 −3-3−3 都可以作为 111 和 999 的等比中项；而 111 和 −9-9−9 不存在等比中项．

等比数列的英文是 Geometric Progression，简称 GP．Geometric 意为「几何的」，正与这里 b=±acb = \pm\sqrt{ac}b=±ac​ 的「几何平均值」（采正值）对应．

有限长度等比数列​

如果题目给定条件「a1,a2,…,aka_1, a_2, \ldots, a_ka1​,a2​,…,ak​ 为等比数列」，我们可以将它 等价 地转化为：

a22=a1a3a32=a2a4⋯ak−12=ak−2aka1a2⋯ak≠0\begin{aligned} {a_2}^2 &= a_1a_3 \\ {a_3}^2 &= a_2a_4 \\ & \cdots \\ {a_{k - 1}}^2 &= a_{k - 2}a_k \\ a_1a_2\cdots a_k & \ne 0 \end{aligned}a2​2a3​2ak−1​2a1​a2​⋯ak​​=a1​a3​=a2​a4​⋯=ak−2​ak​=0​

共 k−2k - 2k−2 个等式和 111 个不等式（也可以理解为 kkk 个不等于 000 的不等式）．

例题 1.2

已知 {an}\{a_n\}{an​}，{bn}\{b_n\}{bn​}，{cn}\{c_n\}{cn​} 成等比数列，证明 a2a^2a2，a2+b2a^2 + b^2a2+b2，(a+c)2(a + c)^2(a+c)2 成等比数列．

例题 1.2 解答

条件等价于 b2=acb^2 = acb2=ac 且 abc≠0abc \ne 0abc=0．

结论等价于 (a2+b2)2=a2(a+c)2(a^2 + b^2)^2 = a^2(a+c)^2(a2+b2)2=a2(a+c)2 且 a2(a2+b2)(a+c)2≠0a^2(a^2 + b^2)(a + c)^2 \ne 0a2(a2+b2)(a+c)2=0．

先证明后边这个不等式（不要忘证了）．因为 a≠0a \ne 0a=0，b≠0b \ne 0b=0，c≠0c \ne 0c=0，所以 a2≠0a^2 \ne 0a2=0，a2+b2≠0a^2 + b^2 \ne 0a2+b2=0．

由于 {bn}\{b_n\}{bn​} 存在，因此 ac>0ac > 0ac>0，a+c≠0a + c \ne 0a+c=0，(a+c)2≠0(a + c)^2 \ne 0(a+c)2=0．不等式证毕．

再证明前面这个等式．(a2+b2)2=(a2+ac)2=a2(a+c)2(a^2 + b^2)^2 = (a^2 + ac)^2 = a^2(a + c)^2(a2+b2)2=(a2+ac)2=a2(a+c)2．等式证毕．

等比数列的增减性​

等差数列中笔者省略了对增减性的讨论——其增减性显然只与 ddd 的正负有关．但在等比数列中会有一些细节，下探讨．

对于公比为 qqq 的等比数列 {an}\{a_n\}{an​}：

  • 当 q<0q < 0q<0 时，{an}\{a_n\}{an​} 为 摆动数列（正负摆动，因此增减摆动）．如果想要探讨 {∣an∣}\{|a_n|\}{∣an​∣} 的增减性，将 qqq 与 a1a_1a1​ 取绝对值再分析即可．
  • 当 q=1q = 1q=1 时，{an}\{a_n\}{an​} 为 常数列．
  • 当 a1>0a_1 > 0a1​>0 且 0<q<10 < q < 10<q<1 时，{an}\{a_n\}{an​} 为 恒正递减数列．
  • 当 a1>0a_1 > 0a1​>0 且 q>1q > 1q>1 时，{an}\{a_n\}{an​} 为 恒正递增数列．
  • 当 a1<0a_1 < 0a1​<0 且 0<q<10 < q < 10<q<1 时，{an}\{a_n\}{an​} 为 恒负递增数列（绝对值 递减）．
  • 当 a1<0a_1 < 0a1​<0 且 q>1q > 1q>1 时，{an}\{a_n\}{an​} 为 恒负递减数列（绝对值 递增）．

可以总结为：

  • q<0q < 0q<0 时，{an}\{a_n\}{an​} 的 正负摆动，增减性摆动，通常不讨论．
  • q>0q > 0q>0 时，{an}\{a_n\}{an​} 恒正 / 恒负．
  • q>0q > 0q>0 时，如果 qqq 足够 贴近 000（0<q<10 < q < 10<q<1），则数列 {an}\{a_n\}{an​} 也会 贴近 000（表现为 绝对值递减）；如果 qqq 距离 000 比较远（q>1q > 1q>1），则数列 {an}\{a_n\}{an​} 会 远离 000（表现为 绝对值递增）．
例题 1.3

已知公比为 qqq 的等比数列 {an}\{a_n\}{an​}，则「a1(q−1)>0a_1(q - 1) > 0a1​(q−1)>0」是「{an}\{a_n\}{an​} 为递增数列」的什么条件？

例题 1.3 解答

显然 a1<0a_1 < 0a1​<0 且 q<0q < 0q<0 满足「a1(q−1)>0a_1(q - 1) > 0a1​(q−1)>0」，但 {an}\{a_n\}{an​} 摆动，而不满足「{an}\{a_n\}{an​} 递增」．

而当「{an}\{a_n\}{an​} 递增」时，a1>0a_1 > 0a1​>0 则 q>1q > 1q>1，a1<0a_1 < 0a1​<0 则 0<q<10 < q < 10<q<1，均满足前者．

因此「a1(q−1)>0a_1(q - 1) > 0a1​(q−1)>0」比「{an}\{a_n\}{an​} 为递增数列」严格弱，前者为后者的 必要不充分条件．

等比数列的性质​

下设 {an}\{a_n\}{an​} 为等比数列．

等比数列性质零​

这是一个排除解的小技巧，基于等比数列较为基本的性质，因此列在这里．

  • q>0q > 0q>0 时，数列 恒负 或 恒正．
  • q<0q < 0q<0 时，数列 正负邻项交替．
  • 无论如何，数列中 下标奇偶性相同 的项，正负性相同．

利用这个性质，我们可以及时排除正负性错误的解．比如已知 a3=5a_3 = 5a3​=5，则 a5a_5a5​ 不可能是负数．如果解出 a5=2a_5 = 2a5​=2 或 a5=−1a_5 = -1a5​=−1，就可以直接排除后者了．

等比数列性质一​

anam=qn−m\dfrac{a_n}{a_m} = q^{n - m}am​an​​=qn−m

该式可以用来 已知任意两项求公比．其变形

an=amqn−ma_n = a_m q^{n - m}an​=am​qn−m

可以用来 已知等比数列的任意一项与公比，求任一另外项．

等比数列性质二（下标守恒定律）​

定义等比数列的若干项相乘、相除（可以带次数，不能带系数）形成的式子为「等比乘除式」.

定义「等比乘除式」两个属性：

  • 次数和：即所有 {an}\{a_n\}{an​} 的 次数 之和．
  • 下标和：即所有 {an}\{a_n\}{an​} 的 下标与次数的积 之和．

如 a92a10\dfrac{{a_9}^2}{a_{10}}a10​a9​2​ 的次数和为 2−1=12 - 1 = 12−1=1，下标和为 9×2+10×(−1)=89 \times 2 + 10 \times (-1) = 89×2+10×(−1)=8．

结论：对于两个「等比乘除式」，如果它们的次数和与下标和均分别相同，则结果一定相同．这就是等比数列的 下标守恒定律．

证明：

=ax1p1ax2p2⋯axkpk=(a1qx1−1)p1(a1qx2−1)p2⋯(a1qxk−1)pk=a1p1+p2+⋯+pkqp1x1+p2x2+⋯+pkxk−(p1+p2+⋯+pk)\begin{aligned} & \phantom = {a_{x_1}}^{p_1}{a_{x_2}}^{p_2}\cdots{a_{x_k}}^{p_k} \\ & = (a_1q^{{x_1} - 1})^{p_1}(a_1q^{{x_2} - 1})^{p_2}\cdots(a_1q^{{x_k} - 1})^{p_k} \\ & = {a_1}^{p_1 + p_2 + \cdots + p_k} q^{p_1x_1 + p_2x_2 + \cdots + p_kx_k - (p_1 + p_2 + \cdots + p_k)} \end{aligned}​=ax1​​p1​ax2​​p2​⋯axk​​pk​=(a1​qx1​−1)p1​(a1​qx2​−1)p2​⋯(a1​qxk​−1)pk​=a1​p1​+p2​+⋯+pk​qp1​x1​+p2​x2​+⋯+pk​xk​−(p1​+p2​+⋯+pk​)​

次数和 p1+p2+⋯+pkp_1 + p_2 + \cdots + p_kp1​+p2​+⋯+pk​ 与下标和 p1x1+p2x2+⋯+pkxkp_1x_1 + p_2x_2 + \cdots + p_kx_kp1​x1​+p2​x2​+⋯+pk​xk​ 确定时，结果确定．

其实「等比乘除式」也可以带系数，此时两个「等比乘除式」的系数还要相同．不过这个只跟等号两边相同系数约分有关，跟等比数列本身没有关系．

等比数列性质三​

对于等比数列 {an}\{a_n\}{an​}，我们生成一个 正整数 kkk，一个 整数 ttt，一个长度为 kkk 的 正整数序列 xxx，一个长度为 kkk 的 实数序列 {cn}\{c_n\}{cn​}，如果我们有

更进一步，对于任意 kkk，我们写出数列
